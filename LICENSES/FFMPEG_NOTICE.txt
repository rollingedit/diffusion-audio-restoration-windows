FFmpeg Notice

Public Windows builds should bundle ffmpeg.exe and ffprobe.exe from BtbN FFmpeg Builds using a Windows x64 LGPL build family.

Approved build source:
https://github.com/BtbN/FFmpeg-Builds

Release evidence must record the exact FFmpeg build filename, source URL, and hash used for the installer. The intended build family is a win64 LGPL ZIP from BtbN, not a GPL or nonfree FFmpeg build.

FFmpeg is licensed separately from RollingEdit A2SB Restorer. The release must preserve the FFmpeg license notice and provide the required LGPL redistribution information for the bundled binaries.
